Question 1: How are Flask and Django different? What does Django provide for you that Flask does not?

Django is a full-stack web framework, which is used for large and complex web application, whereas Flask is a lightweight and extensible web framework. Django provides dynamic webpages and Flask does not.

Question 2: What does REST stand for? When I say something is RESTful, what does that mean?

REST stand for Representational State Transfer. 

Question 3: What does CRUD stand for? For each letter in CRUD, give the associated HTTP method.

CRUD stand for CREATE, READ, UPDATE, AND DELETE.
CREATE: PUT/POST
READ: GET
UPDATE: PUT/POST
DELETE: DELETE

Question 4: What do HTTP 1XX Status Codes mean? HTTP 2xx? HTTP 3xx? HTTP 4xx? HTTP 5xx?

1XX: INFORMATION CODES
2XX: SUCCESS CODES
3XX: REDIRECTION CODES
4XX: CLIENT ERROR CODES
5XX: SERVER ERROR CODES
